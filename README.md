# personal website

This is my personal website for university. It is a simple site where I show a bit about me, my work, and some small projects.

## Pages

- Home: short intro and main skills
- Overview: a summary page with profile, skills, projects, and contact info
- CV: my education, experience, awards, and project table
- Projects: some of the projects I worked on
- Card: a small business card style page
- Chatbot: a simple chatbot page with scripted answers
- Contact: a basic contact form

## What I used

- HTML for the structure
- CSS for the layout, colors, and page design
- JavaScript for the chatbot page
